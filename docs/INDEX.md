## 💡 Pro Tips

1. **Para Windows**: Use PowerShell moderno, não cmd antigo
2. **Para PostgreSQL**: Salve a senha que define na instalação
3. **Para tests**: Execute tudo em ordem do guia de testes
4. **Para produção**: Usar Render para backend (mais fácil)
5. **Para cliente**: Envie CLIENT_QUICKSTART e deixa explorar
6. **Para backup**: Google Drive é nosso padrão (grátis + automático)
