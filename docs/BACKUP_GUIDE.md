## 🎯 O Que é Backup?

Um **backup** é uma **cópia de segurança** dos seus dados. É como fazer uma fotocópia de todos seus documentos importantes para não perder nada.

**Exemplo:**
- Seus agendamentos, clientes, receitas, tudo fica guardado em dois lugares
- Um no servidor principal
- Um em um arquivo de backup

Se algo der errado no servidor principal, você recupera do backup! ✅

## 🛡️ Sistema de Proteção Implementado

### 1️⃣ Backup Automático Diário ✨

**O QUÊ:** Seu sistema faz uma cópia completa dos dados automaticamente

**QUANDO:** Todos os dias às 3 AM (madrugada)

**ONDE:** Em pastas do servidor

**BENEFÍCIO:** Você nunca perde nada! Mesmo que esqueça

```
Segunda-feira às 3 AM → Backup criado ✅
Terça-feira às 3 AM   → Outro backup criado ✅
Quarta-feira às 3 AM  → Mais um created ✅
...
```

### 2️⃣ Backup Manual

**O QUÊ:** Você pode fazer um backup quando quiser

**QUANDO:** Antes de mudanças importantes (deletar clientes, alterar preços, etc)

**COMO:**
1. Abra o app
2. Vá em "💾 Backup"
3. Clique em "📦 Criar Backup Manual"
4. Pronto! Dados salvos

```
Criando Backup Manual
├─ Clientes salvos
├─ Agendamentos salvos
├─ Transações salvas
├─ Relatórios salvos
└─ ✅ Backup concluído!
```

### 3️⃣ Exportação de Dados (CSV/JSON)

**O QUÊ:** Baixar seus dados em formatos abertos

**TIPOS:**

| Tipo | Formato | Usa | Benefício |
|------|---------|-----|-----------|
| Agendamentos | CSV | Excel | Ver cronograma no computador |
| Transações | CSV | Excel | Análise financeira detalhada |
| Tudo | JSON | Backup portátil | Máxima segurança |

**COMO USAR:**

1. **Exportar para Excel:**
   - Vá em "💾 Backup"

   - Arquivo salvo no seu computador

2. **Abrir em Excel:**

   - Visualize todas as informações
   - Imprima se precisar

3. **Compartilhar com Contador:**
