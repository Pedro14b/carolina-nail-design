## 📋 O que você precisa saber (5 minutos)

**Seu aplicativo tem 6 abas principais:**

1. 🏠 **Início** - seu dashboard com resumo do dia
2. 📅 **Agendamentos** - todos seus compromissos
3. 👥 **Clientes** - banco de dados de clientes
4. 🔔 **Notificações** - avisos do sistema
5. 📊 **Relatórios** - sua receita e estatísticas
6. 💾 **Backup e Exportação** - segurança de seus dados

---

## 🚀 Seu Primeiro Dia

### ✅ Passo 1: Fazer Login (ou Criar Conta)

```
Abra o app → Digite seu telefone → Digite sua senha
→ Clique em "Entrar"
```

**💡 Dica**: Se seu telefone tiver leitor de impressão digital, você pode clicar em "Usar Impressão Digital" para login mais rápido!

---

### ✅ Passo 2: Ver o Dashboard

Você verá:
- Quantos agendamentos marcados para hoje
- Quantas notificações você tem
- Acesso rápido para criar novo agendamento

---

### ✅ Passo 3: Criar Seu Primeiro Agendamento

1. Clique na aba **📅 Agendamentos**
2. Clique no botão **"+ Novo Agendamento"** (ou similar)
3. Preencha:
   - **Cliente**: Procure na lista ou crie novo
   - **Serviço**: Escolha (manicure, pedicure, etc)
   - **Data**: Escolha dia desejado
   - **Hora**: Escolha horário
4. Clique **"Confirmar"**

✅ **Pronto!** Seu primeiro agendamento foi criado!

---

### ✅ Passo 4: Adicionar Um Cliente

1. Clique na aba **👥 Clientes**
2. Clique no botão **"+ Novo Cliente"**
3. Preencha:
   - **Nome**: Nome da cliente
   - **Telefone**: Whatsapp (com código de área)
   - **Email**: Opcional
   - **Aniversário**: Opcional (para lembrete)
   - **Endereço**: Opcional
4. Clique **"Salvar"**

✅ **Feito!** Novo cliente cadastrado!

---

## 📅 Seu Segundo Dia

### ✅ Passo 5: Gerenciar Agendamentos

Na aba **📅 Agendamentos**, você pode:

- **Ver todos os agendamentos** em uma lista
- **Filtrar** por status:
  - Pendentes (aguardando confirmação)
  - Confirmados (cliente confirmou)
  - Completados (já fez)
  - Cancelados
- **Clicar em um agendamento** para ver detalhes
- **Editar** data/hora/cliente se precisar
- **Cancelar** se a cliente remarcar
- **Marcar como completo** quando terminar

---

### ✅ Passo 6: Checar Notificações

Na aba **🔔 Notificações**:

- Você receberá lembretes automáticos
- Clique para marcar como lida
- Clique em **"Marcar todas como lidas"** para descartar tudo

**Exemplos de notificações**:
- "Nova cliente agendada para amanhã"
- "30 minutos até próximo agendamento"
- "Backup diário criado com sucesso"

---

### ✅ Passo 7: Conferir Sua Receita

Na aba **📊 Relatórios**:

1. Escolha o período (última semana, mês, ano)
2. Veja:
   - Quanto você ganhou de receita
   - Quanto você gastou
   - Seu lucro total
   - Quantos agendamentos fez
3. Pode exportar em Excel se quiser

---

## 💾 Seu Terceiro Dia

### ✅ Passo 8: Entender o Sistema de Backup

Seu aplicativo **protege automaticamente seus dados**:

- ✅ **Diariamente às 3 AM**: Sistema faz backup automático (você não precisa fazer nada!)
- ✅ **Armazenado com segurança**: Pode estar no seu celular E no Google Drive
- ✅ **Se algo ruim acontecer**: Seus dados podem ser restaurados

**Na aba 💾 Backup e Exportação você pode:**
- Ver quantos backups você tem
- Ver quando foi o último backup
- **Criar um backup manual agora** se quiser (botão "Criar Backup Manual")
- **Exportar dados em Excel** para análise

---

### 📁 Como Exportar em Excel

1. Na aba **💾 Backup e Exportação**
2. Clique em **"Exportar Agendamentos para Excel"** ou
3. Clique em **"Exportar Tudo em JSON"**
4. Arquivo será enviado para seu email ou downloads

**Por que fazer isso?**
- Para ter cópia extra dos seus dados
- Para compartilhar com contador/fiscal
- Para fazer análises completas

## 🆘 Dúvidas Frequentes

### P: Como faço backup se perdi meu telefone?
**R**: Seus dados estão no Google Drive! Você pode reinstalar o app e fazer login.

### P: Posso usar em múltiplos dispositivos?
**R**: Sim! Faça login em outro telefone com mesma senha e terá acesso a todos seus dados.

### P: Quanto tempo demora pra receber um agendamento?
**R**: Instantâneo! Assim que você cria, o sistema notifica seus clientes.

### P: Posso deletar um cliente?
**R**: Sim, mas não recomendamos. É melhor "arquivar" desativando agendamentos futuros.

### P: E se eu quiser adicionar mais um profissional?
**R**: Entre em contato! Podemos configurar usuários adicionais para seus colaboradores.

### P: Como faço para enviar resposta de confirmação?
**R**: Sistema pode enviar SMS/Whatsapp para cliente confirmando. Configuraremos isso em breve!
